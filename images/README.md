# Product Images

Place your product images in this folder (e.g., `ring.jpg`, `necklace.png`, etc.).

When using the **Admin Product Builder** on the website, you can link to these images using their relative paths, such as:
- `images/ring.jpg`
- `images/necklace.png`

Alternatively, you can also paste absolute web URLs (e.g., `https://example.com/image.jpg`) in the product image field.
